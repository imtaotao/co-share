## 什么是 callback
回调只是使用 js 函数的约定的名称。<br>
在 js 中，通常情况下调用一个函数，将立即得到一个结果，这种调用关系称之为同步。这是一种理想的行为。但实际的开发和业务中总会与用户有各种各样的交互行为（也可以成为事件）。这是一些不可预测的行为。这些行为不能立即得到结果，而是需要在一段时间后才能得到结果。例如点击事件，xhr 请求等。
